# Flowbite Svelte Documentation Skill

Use this skill to answer questions about Flowbite Svelte accurately from the official documentation.

## When to use

Use this skill for:

- Flowbite Svelte installation and quickstart questions
- Svelte/SvelteKit integration with Flowbite Svelte
- Component API, props, events, slots, and examples
- Tailwind configuration and theming
- Dark mode, ThemeProvider, colors, customization
- Form controls such as input, select, checkbox, textarea, toggle, radio, file input, range, phone input, number input, search input, floating label, and timepicker
- Plugins such as charts, datatables, and wysiwyg
- Icons, illustrations, utilities, and typography helpers
- Migration or troubleshooting questions where current docs are needed

## Example response pattern

When asked, "How do I make a modal?":

1. Fetch `https://flowbite-svelte.com/llm/components/modal.md`.
2. Summarize the required imports and state management.
3. Provide a minimal `.svelte` snippet.
4. Note any accessibility or slot conventions documented on the page.
